### Summary
This demo demonstrates bar and column charts in a Svelte application using VisuallyJS.

### Components Used
- `BarChartComponent`: Used for rendering horizontal bar charts.
- `ColumnChartComponent`: Used for rendering vertical column charts.

### Component Options
Both components use the following props:
- `options`: A configuration object for the chart.
- `className`: Optional CSS class name for the chart container.

### Stylesheet Requirement
The `visuallyjs.css` stylesheet must be included for the components to function and display correctly.

```css
@import "@visuallyjs/browser-ui/css/visuallyjs.css";
```
